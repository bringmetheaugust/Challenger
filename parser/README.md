### What is this?   

Sites parser for *@carsurfing* project.

***

### How can I deploy it?

##### Localy

 * run `bash local_deploy.sh` to install all dependencies
 * run `bash local_run.sh` to run parser server

##### On *Heroku*
 
 * `git push heroku-parser BRANCH:master`

###### Required global packages:
 * *NodeJS v10+*
 * *yarn v1+*
